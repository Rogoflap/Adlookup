This is the Computer Info page. Here we query the AD Directory for Computer entries.

 

We show the following information.

 
User User's Name (If USERID is in Computer Name)
Userid User's ID (Characters 4-7) of Computer's Name)
Name Computer Name
OS Operating System
Service Pack Operating System Service Pack
Location First 4 Letters of Computer Name
OU OU Folder Computer is in

 

If you do a right click with the mouse on  a row in the grid you will ge tthe following options.

Ping Machine Will Ping the Machine Name
*  SCCM Info Will show SCCM information about machine (IF your userid is in the SCCM Group)
Dameware Machine Will start Dameware Mini Remote session to that machine (If you have Dameware Mini Remote Installed)
Remote Desktop Machine Will start up a remote desktop session to that machine.
Print Grid (All Rows / Selected Rows) Prints out rows in the grid   (Print Preivew first allows you to set up the print (Portraid / Landscape, Border width etc)
Export to Excel (All Rows / Selected Rows) Allows the contents of the grid to be Showin in Excel.
