User info shows USER entries from Active directory.

There are additional Options to
1. Add Your Local Outlook Contacts to the Grid
2. Add User Group Columns  -- Adds Groups Member of info as well as OU Folder User is in.

These entries include:
 
Name Users Full Name as shown in AD
Telephone Number Phone Number
Email Email Address
UserID User ID
Mail Stop Mail stop
Address Physical Address
Acct SAP Account for individual
Company Company
Loc Four Letters after their name in AD  (EALT, CHAS, ect.)
Account Expires Date Account is set to Expire
Last Logon Last Time user logged on
pwdlastset Date/Time password was last set.
lockouttime If user is or was just recently locked out on this domain, then the time this happened.
 * Memberof List of AD Groups this user belongs to
* OU

 OU Folder user is currently in.

If you do a right click with the mouse on  a row in the grid you will ge tthe following options.

Show User Summary Info This will show (Username, FullName, Comment, Account Status, Expire Date, Password last set, Required, Home Directory, Last Logon, Logon Hours Allowed, local group Memberships, Global Groups Memberships for user.
Print Grid (All Rows / Selected Rows) Prints out rows in the grid   (Print Preivew first allows you to set up the print (Portraid / Landscape, Border width etc)
Export to Excel (All Rows / Selected Rows) Allows the contents of the grid to be Showin in Excel.
